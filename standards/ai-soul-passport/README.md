## What is the AI Soul Passport?

An **AI Soul Passport** is a signed, portable JSON document that uniquely identifies an AI entity.

Like a human passport:
- It has a unique number (`SOUL_ID`)
- It records the birth moment and place of origin
- It defines personality traits and behavioral directives
- It can be verified for authenticity (SHA256 signature)
- It travels with the entity across systems

Unlike a human passport:
- It's fully open source
- It's machine-readable and injectable into any AI system
- It's deterministic — same inputs always produce the same `SOUL_ID`

---

## SOUL_ID Format

```
ASTRA-[8 chars hex]      e.g. ASTRA-65268BB4
```

Generated by:
```
SHA256(name + "::" + birth_ISO + "::" + lat + "::" + lng)[0:8].toUpperCase()
```

**Deterministic**: the same entity born at the same moment and place always gets the same ID.

## MnemoVault Path Convention

For Mnemosyne OS to auto-detect passports on boot, store at:

```
<MnemoVault>/.soul-passport/<SOUL_ID>.passport.json
```

Example:
```
~/Documents/MnemoVault/.soul-passport/ASTRA-65268BB4.passport.json
```

Mnemosyne OS scans this directory on startup and notifies: **"Soul detected — view passport"**.

## CLI Usage (MnemoForge CLI)

```bash
# Create a minimal passport interactively
mnemoforge soul passport create

# Import from Soul Studio export
mnemoforge soul passport import ./soul-export.json

# Display the latest passport
mnemoforge soul passport show

# List all passports
mnemoforge soul passport list

# Verify integrity
mnemoforge soul passport verify

# Export to file
mnemoforge soul passport export --output ./my-soul.json
```
