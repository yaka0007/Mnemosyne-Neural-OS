## What Mnemosyne Is

Mnemosyne OS is a **local-first, sovereign AI platform** built around one core premise:

**Memory is not a log. Memory is a state.**

When you remember something important, you don't replay the full conversation. You re-inhabit the state you were in. You recover the intention, the feeling, the context — not the transcript.

Mnemosyne is built to capture, store, and retrieve those states — through a mechanism called the **Narcissus Chronicle**.
