# Chronicle — Neural Coding: The Core Definition

## The Definition

> *"The intention must be present. A connection with an agent that understands
> the situation without a wire between them. The agent must understand the intention —
> but the human must know what they are actually building."*
> — Tony Trochet, 2026-04-05

## What This Means

**Without a wire** — without mechanical, step-by-step instructions.
The agent understands the *context*, the *system*, the *direction* — not just the task.

**The intention** — the human knows WHY they are building what they are building.
Not a feature list. A coherent vision.

**The human must know what they are actually building** — this is not vibe coding.
They are not prompting into the void. They guide from genuine understanding.

## The Difference from Vibe Coding

| Vibe Coding | Neural Coding |
|---|---|
| Human prompts without understanding | Human understands the full system |
| Agent executes without context | Agent understands the intention |
| Result: code that maybe works | Result: a system that makes sense |
| Mechanical connection | Semantic connection |
| Human follows the agent | Human directs, agent amplifies |

## Tonight as Proof

Entire session under relaxed conditions, without notes, without preparation.
Yet every decision was coherent with the global Mnemosyne OS architecture.

Why? Because the intention was already there — embedded in the design.
The CLI invents nothing. It implements what was already conceived.

That is Neural Coding. The code follows the thought. Not the other way around.

## The Rule That Follows

A Neural Coding agent does not ask "what should I do?".
It asks "what is the intention behind this action?".

And the human practicing Neural Coding does not say "do this".
They say "here is where we are going — do you see it?"
